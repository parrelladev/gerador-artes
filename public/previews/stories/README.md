# Previews de Stories

Este arquivo lista os templates de stories e os arquivos de preview (`.png`) usados para exibir as miniaturas na interface.

Regra geral:
- Cada linha da tabela abaixo representa um card de story.
- O arquivo de preview deve ter o nome indicado na coluna **Arquivo de preview** e ficar nesta pasta.
- Se o arquivo não existir, o card mostra um ícone de placeholder.

> Ao criar um novo template ou ajustar um existente, use o mesmo nome de slug + `.png` para a miniatura, quando fizer sentido.
